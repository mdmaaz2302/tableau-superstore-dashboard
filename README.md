## ğŸ“ˆ Dashboard Features

### Key Performance Indicators (KPIs)
- **Total Sales** - Aggregate revenue across all transactions
- **Total Profit** - Bottom-line profitability metrics
- **Total Discounts** - Cumulative discount impact on revenue
- **Profit Margin** - Profitability percentage analysis

### Interactive Visualizations

1. **Category Performance by Region** (Horizontal Bar Chart)
   - Compares sales across Furniture, Office Supplies, and Technology
   - Segmented by regions (Central, East, South, West)
   - Identifies top-performing category-region combinations

2. **Geographic Sales Distribution** (US State Map)
   - Color-coded choropleth map showing sales by state
   - Displays individual state sales values
   - Highlights geographic patterns and regional concentration

3. **Monthly Sales Trend & Regional Forecast** (Line Chart)
   - Historical sales trends from 2018-2022
   - Time-series visualization for trend identification
   - Forecast projection for future sales planning
   - Multi-region comparison with color differentiation
